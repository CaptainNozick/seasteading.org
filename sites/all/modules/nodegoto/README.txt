DESCRIPTION:
------------

This module allows you to define any page or URL as the target, after you:
- create a node
- update a node
- delete a node

It also has the option to send the target page a variable informing from where it came.

If you have the Token module installed you can use dynamic placeholder tokens in your paths.

It works with CCK.

Uses:
•	Congratulations page.
•	Guide the user on creating node after node
•	etc...


INSTALLATION
------------

Copy the similar directory to your modules directory.
(example.org/modules/nodegoto)

Activate the module in administer > modules.

Configure the path for each node in  admin/settings/nodegoto
